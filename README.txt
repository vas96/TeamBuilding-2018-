Отже, всім відома ситуація при створенні стартапу: ідея є, засоби, інтсрументи, можливості є, а людей нема...

Дана програма призначена для вирішення цієї ситуації.
Користувач - програміст, дизайнер, аніматор (одним словом людина, яка володіє деякими навичками) - реєструється в програмі, переглядає запропоновані проекти, подає заявки на участь у них, переписується з іншими користувачами, або створює власний проект, вносить інформацію про навички, якими повинна володіти людина для участі у проекті і розвиває свій проект.

Реалізовано:
-додавання проектів
-реєстрація
-сторінка проекта
-пошук проекта
-редагування персональних даних користувача

На майбутнє:
-чат
-пошук за категоріями
-пошук користувача
-поступова підгрузка проектів
-архітектура клієнт-сервер
